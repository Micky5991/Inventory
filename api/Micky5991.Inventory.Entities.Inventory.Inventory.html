    InsertItem(IItem, Boolean)

    Tries to insert the given item into this inventory.

    • If the given item is already in any other inventory, it will be removed from the old one first.

    • If an item, that is ready to be merged with the given item exists in this inventory, it will be merged into the first existing item and the given item will be disposed.
